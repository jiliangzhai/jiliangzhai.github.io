时间与日期的处理。

这篇博客的内容主要是IOS开发中可能会遇到的时间与日期的处理，虽不复杂但偶尔还会给你添点小麻烦，本文中涉及到的类包括：NSDate、NSTimeZone、NSDateComponents以及NSCalendar。博客中不会将所有方法都叙述一遍，主要是一些常用方法及细节。

NSDate

NSDate *date = [NSDate date];
//最常用的方法，返回的是计算机当前时间，需要注意的是返回的是GMT时间。

也可以利用时间偏移量进行NSDate的初始化，方法如下：

NSDate *date = [NSDate dateWithTimeInterval:(NSTimeInterval) sinceDate:(nonnull NSDate *)];
//以参数日期为基准进行偏移，得到目标时期。
NSDate *date = [NSDate dateWithTimeIntervalSince1970:(NSTimeInterval)];
//以1970-01-01 00:00:00 ＋0000为基准进行偏移得到目标日期。
NSDate *date = [NSDate dateWithTimeIntervalSinceNow:(NSTimeInterval)];
//以当前时间为基准进行偏移得到目标日期。
NSDate *date = [NSDate dateWithTimeIntervalSinceReferenceDate:(NSTimeInterval)];
//以参考时间2001-01-01 00:00:00为基准进行偏移得到目标时间。

两个日期之前可以进行是否相同以及较早/较晚的判断方法如下：

//判断两个日期是否相同。
BOOL isEqual = [date1 isEqualToDate:date2];
//返回较晚的那个日期
NSDate *laterDate = [date1 laterDate:date2];
//返回较早的那个日期
NSDate *earlierDate = [date1 earlierDate:date2];
//compare方法，根据返回的是NSOrderedSame/NSOrderedAscending/NSOrderedDescending判断两日期的差别。
NSComparisonResult result = [date1 compare:date2];

日期之间还可以进行时间间隔的计算，主要方法如下：

NSInteger interval = [date1 timeIntervalSince1970];
NSInteger interval = [date1 timeIntervalSinceDate:date2];
NSInteger interval = [date1 timeIntervalSinceNow];
NSInteger tnterval = [date1 timeIntervalSinceReferenceDate];
//这些方法与以时间偏移量进行日期的初始化过程一一对应，为反过程。

NSDateComponents

NSDateComponents顾名思义包含了组成日期的每个部分，例如年、月、日、时、分等，这样我们可以利用个部分去组成一个日期，也可以从一个日期中获得我们想得到的部分信息。除了以上部分外，该类还包含了时区，历法信息。

说到NSDateComponents，我们必须提到NSCalendar类，该类提供的两个方法就可以实现我们上面提到的两个过程及由日期各部分获得日期和由日期得到各部分，代码如下：

 NSCalendar *calendar = [NSCalendar currentCalendar];
//NSCalendar *calendar = [NSCalendar autoupdatingCurrentCalendar];这是两个类方法，区别是前者获得的结果会存在缓存中即使更改了系统历法也不会改变，后者在更改后会获得更改后的历法。
NSDateComponents *components = [[NSDateComponents alloc] init];
[components setYear:16];
[components setMonth:03];
[components setDay:24];
//设置NSDateComponents
NSDate *date = [calendar dateFromComponents:components];
//根据components获得date
NSDateComponents *newComponents = [calendar components:NSCalendarUnitYear|NSCalendarUnitMonth|NSCalendarUnitDay fromDate:date];
//由date获得components

NSDateComponens还可以作为偏移，在基准日期的基础上获得新的日期，或者得到两个日期之间的差值以NSDateComponents表示，两个过程如下：

NSCalendar *calendar = [NSCalendar currentCalendar];
NSDateComponents *components = [[NSDateComponents alloc] init];
[components setYear:16];
[components setMonth:03];
[components setDay:24];
//设置NSDateComponents
NSDate *date = [calendar dateByAddingComponents:components toDate:[NSDate date] options:0];
//以components作为差值，获得新的日期。
NSDateComponents *newComponents = [calendar components:NSCalendarUnitYear|NSCalendarUnitMonth|NSCalendarUnitDay fromDate:[NSDate date] toDate:date options:0];
//获得两个日期之间的差值以components表示

NSDateFormatter

最主要的方法就是stringFromDate和dateFromString，可以把日期按照设定的格式输出字符串，或者把字符串按照设定的格式转换为日期。方法如下：

NSDate *date = [NSDate date];
NSDateFormatter *formatter = [[NSDateFormatter alloc] init];
[formatter setDateFormat:@"yyyy-MM-dd HH-mm-ss"];
NSString *dateStr = [formatter stringFromDate:date];
//日期转字符串
NSDate *strToDate = [formatter dateFromString:dateStr];
//字符串转日期

NSDateFormatter是包含时区信息的，更改时区使用 setTimeZone方法 。

NSTimeZone

NSTimeZone主要的方法就是进行时区间的转化，以避免时区不同带来的问题，主要方法ruxia：

NSTimeZone *zone = [NSTimeZone systemTimeZone];
NSString *str = [zone abbreviation];
//系统时区

NSTimeZone *localZone = [NSTimeZone localTimeZone];
NSString *str2 = [localZone abbreviation];
//当地时区
NSDate *date = [NSDate date];
NSInteger offSet = [zone secondsFromGMTForDate:date];
NSDate *localDate = [date dateByAddingTimeInterval:offSet];
//根据时区差值更新日期

NSArray *timeZones = [NSTimeZone knownTimeZoneNames];
//获得时区名称
